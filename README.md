# LiHuizi Portfolio

Dark, restrained personal portfolio for LiHuizi, built with React, Vite, TypeScript, and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:3000/`.

## Build

```bash
npm run build
```

## Structure

- `src/App.tsx` - page composition and sections
- `components/` - reusable visual components
- `data/projects.ts` - project and content data
- `styles.css` - global visual system
